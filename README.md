# Data Science Learning Workspace

This workspace contains curated learning material for data science, machine learning, statistics, and data visualization. Each folder is organized by topic and includes Jupyter notebooks, datasets, and project examples.

## Folder Structure

- `machine learning/`
  - `machine learning part 1/`
    - `1_basic_terminology.ipynb` - Introduction to core machine learning terms and concepts.
    - `2_Steps_involved_in_creating_ML.ipynb` - Walkthrough of the end-to-end machine learning process.
    - `3_Project_1.ipynb` - Hands-on ML project with real datasets.
    - `4_Project_2.ipynb` - Advanced ML project example.
    - `heart.csv` - Dataset for classification projects.
    - `insurance.csv` - Dataset for regression projects.
  
  - `machine learning part 2/` - Supervised Learning Fundamentals
    - `1_Regression.ipynb` - Linear regression theory, formulas, and real-time house price prediction example with visualization.
    - `2_Essentials.ipynb` - Train/test split, MSE, R², and overfitting concepts with detailed comparisons.
    - `3_Project_1.ipynb` - Applied regression project with real data analysis.
    - `4_Project_2.ipynb` - Advanced regression techniques and optimization.
    - `ford.csv` - Dataset for regression examples.
    - `insurance.csv` - Dataset for model evaluation demonstrations.
  
  - `machine learning part 3/` - Classification Algorithms (Comprehensive with Satirical Explanations)
    - `1_Basics.ipynb` - Classification fundamentals: confusion matrix, accuracy, precision, recall, F1-score with worked examples.
    - `2_Logistic_Regression.ipynb` - Binary classification using sigmoid function with "magic squasher" analogy and student exam prediction.
    - `3_KNN.ipynb` - K-Nearest Neighbors with "Finding Your Tribe" party analogy and movie genre classification.
    - `4_Naive_Bayes.ipynb` - Probabilistic classification with belief update detective analogy, spam detection, and movie review examples.
    - `5_Decision_Tree.ipynb` - Tree-based classification with entropy/information gain concepts and loan approval decision rules.
    - `6_SVM.ipynb` - Support Vector Machines with margin maximization and kernel trick explanations.
    - `7_Project_1.ipynb` - Classification project applying multiple algorithms.
    - `8_Project_2.ipynb` - Advanced classification challenges and comparisons.
    - `images/` - Centralized folder containing all 17 satirical and interactive diagrams for Part 3 algorithms.
    - `heart.csv` - Dataset for classification demonstrations.

- `Statistics/`
  - `ANNOVA_test.ipynb` - Analysis of variance (ANOVA) explained and demonstrated.
  - `ChiSquare.ipynb` - Chi-square test concepts and examples.
  - `outliers.ipynb` - Identifying and handling outliers in data.
  - `t-test.ipynb` - Single-sample and paired t-test examples.
  - `two_sample_T-test.ipynb` - Two-sample t-test comparisons.
  - `Ztest.ipynb` - Z-test fundamentals and examples.

- `vizualizations ( matplotlib, seaborn, cufflinks )/`
  - `1_completeMatplotLib.ipynb` - Complete Matplotlib visualization guide.
  - `2_distributionPlots.ipynb` - Distribution plotting techniques.
  - `3_categoricalPlots.ipynb` - Categorical data visualization methods.
  - `4_matrixPlot.ipynb` - Matrix plots and heatmaps.
  - `5_RegressionPlots.ipynb` - Regression and trend visualizations.
  - `IPL_Capstone_project.ipynb` - IPL dataset visualization capstone project.
  - `IPL.csv` - Dataset used by the IPL visualization project.

- `pythonBOOK.pdf` - A PDF book related to Python programming.

## How to use this workspace

1. Open this folder in VS Code or GitHub Desktop.
2. Navigate to a notebook you want to explore.
3. Open the notebook and run cells interactively to understand the concepts.

## Recommended learning path

**Part 1: Foundations**
1. Start with `machine learning/machine learning part 1/1_basic_terminology.ipynb` - Learn ML basics.
2. Continue to `machine learning/machine learning part 1/2_Steps_involved_in_creating_ML.ipynb` - Understand the ML pipeline.
3. Practice with `machine learning/machine learning part 1/3_Project_1.ipynb` - Apply concepts to real data.
4. Advance to `machine learning/machine learning part 1/4_Project_2.ipynb` - Build more complex models.

**Part 2: Regression & Model Evaluation**
5. Study `machine learning/machine learning part 2/1_Regression.ipynb` - Master linear regression theory and applications.
6. Learn `machine learning/machine learning part 2/2_Essentials.ipynb` - Understand train/test splits, MSE, R², and overfitting.
7. Apply with `machine learning/machine learning part 2/3_Project_1.ipynb` and `4_Project_2.ipynb` - Real regression projects.

**Part 3: Classification Algorithms (Interactive & Satirical)**
8. Begin with `machine learning/machine learning part 3/1_Basics.ipynb` - Classification fundamentals and metrics.
9. Study `machine learning/machine learning part 3/2_Logistic_Regression.ipynb` - Binary classification with sigmoid function.
10. Explore `machine learning/machine learning part 3/3_KNN.ipynb` - Instance-based learning.
11. Understand `machine learning/machine learning part 3/4_Naive_Bayes.ipynb` - Probabilistic classifiers.
12. Master `machine learning/machine learning part 3/5_Decision_Tree.ipynb` - Tree-based decisions with entropy.
13. Learn `machine learning/machine learning part 3/6_SVM.ipynb` - Margin maximization and kernel methods.
14. Practice with `machine learning/machine learning part 3/7_Project_1.ipynb` and `8_Project_2.ipynb` - Multi-algorithm projects.

**Statistics & Visualization**
15. Review statistical inference in the `Statistics/` notebooks for hypothesis testing concepts.
16. Finish with visualization notebooks under `vizualizations ( matplotlib, seaborn, cufflinks )/` to master data presentation.

- **Comprehensive Explanations**: Part 3 notebooks feature interactive, satirical explanations with real-world analogies (e.g., "magic squasher" for sigmoid, "Finding Your Tribe" for KNN, "belief updates" for Naive Bayes).
- **Worked Examples**: Each algorithm includes step-by-step numerical worked examples with actual calculations and predictions.

## Key Features of This Workspace

- **Interactive Learning**: Run Jupyter notebooks cell-by-cell to understand concepts progressively.
- **Theory & Practice**: Each algorithm combines mathematical foundations with practical worked examples.
- **Visual Learning**: Comprehensive diagrams, visualizations, and plots throughout all notebooks.
- **Satirical Approach**: Complex concepts explained with humor and memorable analogies for better retention.
- **Real Datasets**: Learn using actual datasets (heart.csv, insurance.csv, ford.csv, IPL.csv) for practical experience.
- **Progressive Complexity**: Content flows from fundamentals (Part 1) → regression (Part 2) → classification (Part 3) → statistics & visualization.
