# Data Science Learning Workspace

This workspace contains curated learning material for data science, machine learning, statistics, and data visualization. Each folder is organized by topic and includes Jupyter notebooks, datasets, and project examples.

## Folder Structure

- `machine learning/`
  - `machine learning part 1/`
    - `1_basic_terminology.ipynb` - Introduction to core machine learning terms and concepts.
    - `2_Steps_involved_in_creating_ML.ipynb` - Walkthrough of the end-to-end machine learning process.
    - `Project_1.ipynb` - A hands-on machine learning project using a real dataset.
    - `insurance.csv` - Dataset used in one or more machine learning examples.

- `Statistics/`
  - `ANNOVA_test.ipynb` - Analysis of variance (ANOVA) explained and demonstrated.
  - `ChiSquare.ipynb` - Chi-square test concepts and examples.
  - `outliers.ipynb` - Identifying and handling outliers in data.
  - `t-test.ipynb` - Single-sample and paired t-test examples.
  - `two_sample_T-test.ipynb` - Two-sample t-test comparisons.
  - `Ztest.ipynb` - Z-test fundamentals and examples.

- `vizualizations ( matplotlib, seaborn, cufflinks )/`
  - `1_completeMatplotLib.ipynb` - Complete Matplotlib visualization guide.
  - `2_distributionPlots.ipynb` - Distribution plotting techniques.
  - `3_categoricalPlots.ipynb` - Categorical data visualization methods.
  - `4_matrixPlot.ipynb` - Matrix plots and heatmaps.
  - `5_RegressionPlots.ipynb` - Regression and trend visualizations.
  - `IPL_Capstone_project.ipynb` - IPL dataset visualization capstone project.
  - `IPL.csv` - Dataset used by the IPL visualization project.

- `pythonBOOK.pdf` - A PDF book related to Python programming.

## How to use this workspace

1. Open this folder in VS Code or GitHub Desktop.
2. Navigate to a notebook you want to explore.
3. Open the notebook and run cells interactively to understand the concepts.

## Recommended learning path

1. Start with `machine learning/machine learning part 1/1_basic_terminology.ipynb`.
2. Continue to `machine learning/machine learning part 1/2_Steps_involved_in_creating_ML.ipynb`.
3. Explore the project in `machine learning/machine learning part 1/Project_1.ipynb`.
4. Review statistical inference in the `Statistics/` notebooks.
5. Finish with the visualization notebooks under `vizualizations ( matplotlib, seaborn, cufflinks )/`.
